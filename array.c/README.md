# 📚 Arrays in C

A collection of programs demonstrating fundamental operations on **One-Dimensional** and **Two-Dimensional Arrays** in C.

---

## 📂 Files

| File         | Description                          |
| :----------- | :----------------------------------- |
| `1D_Array.c` | Operations on One-Dimensional Arrays |
| `2D_Array.c` | Operations on Two-Dimensional Arrays |

---

## ✨ Topics Covered

### 🔹 One-Dimensional Array

* Declaration
* Initialization
* Display Array
* Access Element
* Update Element
* Traverse Array
* Find Maximum
* Find Minimum
* Find Sum
* Find Average

### 🔹 Two-Dimensional Array

* Declaration
* Initialization
* Display Matrix
* Access Element
* Update Element
* Traverse Matrix
* Find Row Sum
* Find Column Sum
