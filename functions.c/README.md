# 🔧 Functions in C

This folder contains programs demonstrating the concepts of **functions in C programming**. It covers predefined functions, user-defined functions, recursion, and parameter passing techniques.

---

## 📂 Files Included

| File                          | Description                                         |
| ----------------------------- | --------------------------------------------------- |
| 📌 `predefined_function.c`    | Programs using built-in library functions           |
| 🛠️ `user_defined_function.c` | Demonstrates user-defined functions and their types |
| 🔁 `recursive_function.c`     | Programs based on recursion and parameter passing   |
